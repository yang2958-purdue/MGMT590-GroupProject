# 🚀 Quick Start Guide

## Get Running in 3 Minutes!

### Step 1: Prepare Icons (30 seconds)
Open `generate-icons.html` in your browser → Icons download automatically ✅

**OR** just use the existing SVG icons (they work fine!)

### Step 2: Load Extension (1 minute)
1. Open Chrome
2. Go to: `chrome://extensions/`
3. Toggle **Developer mode** ON (top right)
4. Click **"Load unpacked"**
5. Select folder: `C:\DEVOPS FOLDER\MGMT590-GroupProject`
6. ✅ Done!

### Step 3: Try It Out! (1 minute)
1. **Click the extension icon** in Chrome toolbar
   - Side panel opens on left
   
2. **Upload a resume** (optional)
   - Click or drag PDF/DOCX file
   
3. **Go to any website** (like google.com)

4. **Press Right Arrow key (→)**
   - Watch the magic! 🎯
   
5. **Press Right Arrow (→) again** to stop

---

## That's It! 🎉

**Keyboard Shortcut:** Right Arrow (→) = Toggle ON/OFF

**Status:** Check the side panel to see if cursor control is ON or OFF

## Common Questions

**Q: Cursor not moving?**  
A: Make sure you're on a regular website (not chrome:// pages)

**Q: Side panel won't open?**  
A: Click the extension icon in toolbar (may be in ⋮ menu)

**Q: How do I stop it?**  
A: Press Right Arrow (→) key again

**Q: Is my resume saved?**  
A: Yes! It's stored locally in Chrome and persists across sessions
